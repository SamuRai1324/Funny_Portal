Пользовательское соглашение

Дата последнего обновления: 15.01.2026

Настоящее Пользовательское соглашение (далее — «Соглашение») регулирует отношения между Администрацией веб-сервиса MyApp (далее — «Сервис», «Платформа») и физическим лицом (далее — «Пользователь»), использующим функциональные возможности Сервиса. Регистрация на Платформе означает полное и безоговорочное принятие настоящего Соглашения.

1. Общие положения

1.1. Сервис MyApp представляет собой социальную платформу, предоставляющую возможности для публикации контента, общения, обмена информацией и взаимодействия между пользователями.

1.2. Использование Сервиса допускается только при условии согласия с настоящим Соглашением. Если Пользователь не согласен с условиями, он обязан немедленно прекратить использование Сервиса.

1.3. Администрация оставляет за собой право изменять условия Соглашения без предварительного уведомления. Актуальная версия всегда доступна на данной странице.

1.4. Продолжение использования Сервиса после внесения изменений означает согласие Пользователя с новой редакцией Соглашения.

2. Регистрация и учетная запись

2.1. Для доступа к полному функционалу Сервиса необходима регистрация с указанием достоверных данных.

2.2. Пользователь несет полную ответственность за:

  • Сохранность и конфиденциальность логина и пароля
  • Все действия, совершенные под его учетной записью
  • Достоверность предоставленной информации

2.3. Один пользователь может иметь только одну учетную запись. Создание мультиаккаунтов запрещено.

2.4. Администрация вправе заблокировать или удалить учетную запись без объяснения причин при нарушении условий Соглашения.

3. Правила использования Сервиса

3.1. Пользователь обязуется использовать Сервис добросовестно и в соответствии с его назначением.

3.2. При использовании Сервиса запрещается:

  • Публикация контента, нарушающего законодательство Российской Федерации
  • Размещение материалов экстремистского, порнографического или насильственного характера
  • Оскорбления, травля, дискриминация других пользователей
  • Спам, массовые рассылки, навязчивая реклама
  • Распространение вредоносного программного обеспечения
  • Попытки несанкционированного доступа к данным других пользователей
  • Публикация чужого контента без указания авторства
  • Использование автоматизированных средств для взаимодействия с Сервисом
  • Любые действия, нарушающие работоспособность Сервиса

3.3. Администрация вправе удалять любой контент без предварительного уведомления и объяснения причин.

4. Публикация контента

4.1. Пользователь может публиковать текстовые записи, видеоматериалы, комментарии и создавать каналы в соответствии с функционалом Сервиса.

4.2. Публикуя контент, Пользователь подтверждает наличие всех необходимых прав на данный контент.

4.3. Пользователь предоставляет Администрации неисключительную безвозмездную лицензию на хранение, отображение и распространение опубликованного контента в рамках функционала Сервиса.

4.4. Администрация не несет ответственности за содержание пользовательского контента.

5. Система коммуникации

5.1. Сервис предоставляет функционал общего чата и системы комментариев для коммуникации между пользователями.

5.2. Все сообщения в общем чате являются публичными и доступны всем авторизованным пользователям.

5.3. Администрация не осуществляет предварительную модерацию сообщений, но оставляет за собой право удалять контент, нарушающий правила Сервиса.

5.4. Запрещается использование системы коммуникации для спама, мошенничества и иных противоправных действий.

6.3. Пользователь сохраняет авторские права на созданный им контент.

7.1. Администрация собирает и обрабатывает персональные данные Пользователей в соответствии с требованиями Федерального закона № 152-ФЗ «О персональных данных».

7.2. Собираемые данные включают:

  • Логин и адрес электронной почты

7.3. Персональные данные не передаются третьим лицам за исключением случаев, предусмотренных законодательством.

7.4. Пользователь вправе запросить удаление своих персональных данных, направив запрос Администрации.

10.1. Настоящее Соглашение вступает в силу с момента регистрации Пользователя и действует бессрочно.

10.2. Признание отдельных положений Соглашения недействительными не влечет недействительности остальных положений.

10.3. Бездействие Администрации при нарушении Соглашения не лишает её права предпринять соответствующие действия позднее.

10.4. По всем вопросам, не урегулированным Соглашением, стороны руководствуются действующим законодательством Российской Федерации.
